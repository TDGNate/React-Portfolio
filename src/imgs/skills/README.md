## Demo Code For Icons

```html

	<a href="{ LINK TO URL }" target="blank" className="skill-anchor">
		<img src="{" THE SKILL } alt="{ SKILL } icon" className="skill-img" />
	</a>
	<p className="skill-title ">{ SKILL TITLE }</p>
</div>
```
